# Taller de POO - Figuras Geométricas
**Estudiante:** Nasly

## Descripción
Este proyecto implementa los conceptos de Programación Orientada a Objetos en Python para el cálculo de áreas y perímetros.

### Conceptos Aplicados:
* **Herencia:** Uso de una clase base `Figura`.
* **Polimorfismo:** Sobrescritura de métodos `area()` y `perimetro()`.
* **Encapsulamiento:** Atributos privados (`__`) con `@property` y `@setter`.
* **Validaciones:** Control de datos de entrada (evita radios negativos).

## Evidencia de Ejecución
El código fue probado en PyCharm y Google Colab, asegurando el cumplimiento de la normativa PEP8.
